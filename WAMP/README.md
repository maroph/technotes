# WAMP
WAMP ist eine [XAMPP](https://www.apachefriends.org/)
ähnliche Umgebung für Windows.

Auf einem **W**indows System werden dabei

* **Apache** Web Server
* **M**ariaDB
* **P**HP

so konfiguriert, dass alle Daten in einem Verzeichnis abgelegt
sind. Keine der Komponenten wird installiert oder als Service
betrieben.
